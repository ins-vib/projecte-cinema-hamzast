El sistema de códigos promocionales permite al administrador crear códigos de descuento desde `/admin/promos`, definiendo el nombre del código, el porcentaje de descuento (1-100%), las fechas de validez y el número máximo de usos totales. El cliente puede introducir un código en el carrito, y el sistema lo valida en tiempo real comprobando que exista, esté activo, no haya caducado, no haya agotado sus usos y que ese usuario no lo haya usado ya anteriormente. Si pasa todas las validaciones, el carrito muestra el precio original, el descuento aplicado y el total final. Al hacer checkout, el descuento se aplica al pedido, se registra el uso en la tabla `PromoUsage` con el usuario, el ahorro exacto y la fecha, y se incrementa el contador del código, todo dentro de una misma transacción. El admin también puede activar o desactivar códigos sin borrarlos, editarlos y consultar el historial completo de quién ha usado cada código y cuánto se ha ahorrado.
